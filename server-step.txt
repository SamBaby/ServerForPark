***安裝TERMUX***
1.安裝termux-app_v0.118.0+2f40df9-apt-android-5-github-debug_armeabi-v7a.apk
2.termux-setup-storage
3.echo "deb https://packages.termux.dev/termux-main-21/ statble main" > $PREFIX/etc/apt/sources.list
4.echo "deb https://termux.dev/game-packages-21-bin games stable" >  $PREFIX/etc/apt/sources.list.d/game.list
5.echo "deb https://termux.dev/science-packages-21-bin science stable" > $PREFIX/etc/apt/sources.list.d/science.list
6.pkg upgrade
7.第一個輸入Y，之後都輸入N

***將ServerForPark資料夾底下的檔案複製到sdcard下面***
camera_server.py/parking_lots.sql/httpd.conf/htdocs

***安裝APACHE伺服器+mysql資料庫****
可以參考https://www.youtube.com/watch?v=dmhJQ4zs_WY
1.pkg install php-apache -y
2.pkg install mariadb -y
3.cp /storage/sdcard/httpd.conf $PREFIX/etc/apache2/
4.cd /storage/sdcard/
5.匯入SQL資料
 (1)mysqld
 (2)開新分頁，並且輸入mysql
 (3)ALTER USER 'root'@'localhost' IDENTIFIED VIA mysql_native_password USING PASSWORD('root');
 (4)CREATE DATABASE parking_lots;
 (5)use parking_lots;
 (6)source parking_lots.sql;
6.pkg install python -y
7.pip install requests

開啟伺服器
1.mysqld
2.rm $PREFIX/var/run/apache2/httpd.pid
2.apachectl start
3.python /storage/sdcard/camera_server.py
關閉伺服器
1.到設定的應用程式強制關閉termux
